## 프로젝트 개요

본 프로젝트는 Unity 기반 **Donkeycar 시뮬레이터**와 연동하여 자율주행 데이터를 효율적으로 확인·정제하고 AI 모델 학습을 관리할 수 있는 Windows UI 애플리케이션입니다.

### 전체 동작 흐름

```
시뮬레이터 주행 → 데이터 수집 (catalog JSON) → C# UI 데이터 정제 → AI 모델 학습 → 자율주행 검증
```

---

## 주요 기능

- 📂 주행 데이터(이미지 + catalog JSON) 로드 및 시각화
- 🔍 OpenCV 기반 차선 인식 결과 실시간 표시
- 🗑️ 불량 프레임 삭제 및 복구 (단일 / 구간 / 다중 선택)
- 🎛️ 네온 계기판 바늘 애니메이션 (속도 / 조향각 동기화)
- 🤖 Python CNN/LSTM 모델 학습 실행 및 실시간 로그 모니터링
- 📊 학습 Loss 그래프 실시간 시각화
- 🏁 AI 예측값과 실제값 비교 분석 (파일럿 아레나)

### 2. 학습 탭

> Python AI 모델 학습을 실행하고 결과를 모니터링하는 화면입니다.

<img width="1513" height="978" alt="스크린샷 2026-06-08 205203" src="https://github.com/user-attachments/assets/51caef5b-326f-4ebf-8ef4-88d1d05c19e3" />


#### 학습 설정 및 실행
- **모델 선택**: CNN / LSTM 중 학습 모델 선택
- **가상환경 선택**: WSL conda 환경 자동 감지 및 선택
- **필터링 옵션**: angle=0 또는 throttle≤0 데이터 제외 옵션
- **Train 버튼**: 클릭 시 데이터 전처리 → 흑백 이미지 생성 → Python 학습 프로세스 실행
- **학습 중단 버튼**: 실행 중인 학습 프로세스 즉시 중단

#### 실시간 모니터링
- **로그 창**: Epoch별 진행 상황, Loss 수치, 오류 메시지 실시간 출력
- **진행률 프로그레스바**: 전체 학습 진행률 (%) 표시
- **Loss 그래프**: Train Loss / Val Loss 변화 추이를 실시간 꺾은선 그래프로 시각화

## 5. 구현 단계 로드맵

전체 개발 단계는 아래와 같이 총 8단계의 순차적 미션으로 진행됩니다.

1. catalog 파일 읽기: JSON 파일 구조를 분석하고 데이터를 파싱하는 기본 인프라 구축
2. 이미지 + 데이터 표시: UI 패널에 이미지 영상과 제어 수치를 동기화하여 매칭 출력
3. 프레임 슬라이더 구현: 전체 프레임을 자유롭게 탐색할 수 있는 트랙바 핸들러 연동
4. 그래프/자동재생 구현: 주행 데이터 수치의 그래프화 및 영상 형태의 타이머 재생 기능 개발
5. 데이터 필터링: 특정 조건이나 범위의 데이터를 선별할 수 있는 필터 조건부 처리
6. 데이터 삭제 기능: 불량 데이터를 물리적/논리적으로 제외하는 정제 기능 구현
7. 리스트 선택: 데이터셋 리스트 관리 및 개별 셋 선택 인터페이스 최적화
8. 학습 실행 및 연동: C# UI 환경에서 Python 백엔드 엔진을 호출하여 전체 학습 파이프라인 완성

## 2. 탭별 최종 구현 기능 및 UI/UX 확정 사항

### 탭 1 — Data Manager (데이터 확인 및 정제)
* **이미지 및 엣지 표시:** 주행 원본 이미지 출력 및 OpenCV 기반 래인(차선) 인식 결과 시각화

* **계기판 시각화 동기화:**
  * 속도(Throttle): 네온 그린 스타일의 자동차 계기판 바늘 드로잉 연동
  * 조향각(Angle): 네온 블루 스타일의 컴파스 나침반 형태 바늘 회전 연동 (데이터 매칭 완벽 보정)
  * **[추가]** 예측 경로 계산 및 경로 표시 온/오프 기능 탑재
* **프레임 화질 조절 기능**

### 탭 2 — Training (AI 모델 학습 관리)
* **모델 선택 및 관리:** Linear 등 학습 모델 타입을 고르는 콤보박스 동기화
* **학습 및 제어:** Train 시작 버튼 및 학습 즉시 중단 버튼 구현
* **실시간 학습 모니터링:** * 하이테크 스타일로 리뉴얼된 로그 리스트 창 및 학습 진행률 프로그레스 바 출력
  * 주석 및 정제가 완료된 최종 데이터셋만 선별하여 흑백 이미지 전처리 후 학습 연동 (전처리 실패 목록 로그 출력)
* **Loss 그래프 연동:** 에폭(Epoch) 변화에 따른 로스(Loss)율 감소 추이를 실시간 꺾은선형(Line) 네온 그래프로 시각화
* **[추가] AI 학습 확인 기능 (이상 탐지 및 비교):**
  * 이미지 및 카탈로그 인덱스를 매칭하여, 기존 원본 데이터셋의 수치와 AI가 실시간으로 예측한 결과(속도, 앵글)를 화면에 따로 표시하고 상호 비교할 수 있는 모니터링 기능 구축
